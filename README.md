# Puppy-8 ##[v.0.1.0] 
+1 CHIP-8 Implementation

## Usage
Building with Maven:
```
mvn install
```
after this, execute in command line:
```
java -jar <puppy8-name>.jar <rom-name>.ch8
```
Controls:
```
Chip8 Key   Keyboard
---------   ---------
 1 2 3 C     1 2 3 4
 4 5 6 D     q w e r
 7 8 9 E     a s d f
 A 0 B F     z x c v
```
Game roms:
https://johnearnest.github.io/chip8Archive/
https://www.zophar.net/pdroms/chip8/chip-8-games-pack.html
